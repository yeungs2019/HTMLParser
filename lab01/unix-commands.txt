11. Explain what |, <, >, and >> do when used on the command prompt.
A: < is used to redirect input from disk file. > is used to redirect output to adisk file.   >> is used when a user wants to append output to existing file. | is a pipe like function that is used to redirect output from one program to another.  *from askUbuntu.  

12. What is the difference between * and ? in filename wildcards?
A: * tracks all possible file places, while ? tracks only one specific one.
